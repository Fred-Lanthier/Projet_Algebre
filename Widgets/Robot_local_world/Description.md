Description

Cette activité p5.js doit illustré le changement de repère entre une base globale et une base locale à travers le défi d’un robot voyageur. Le robot doit atteindre une cible définie dans le repère monde, mais il ne peut recevoir que des commandes exprimées dans son propre repère local.

L’utilisateur peut déplacer le robot, déplacer la cible, modifier l’orientation du robot, puis tester une commande locale pour observer si elle permet d’atteindre la cible. L’activité permet de visualiser concrètement le lien entre un vecteur exprimé dans le repère monde et ses coordonnées dans le repère local du robot à l’aide d’une matrice de rotation et de sa transposée.
